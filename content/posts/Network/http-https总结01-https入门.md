# 关键点：
1. 非对称加密: 公钥加密，私钥解密；私钥签名，公钥验签。
2. 对称加密
3. 消息摘要(hash算法),md5,sha256等
4. 数字证书机制
(前三点也是区块链技术中的重要点)
